# Estimation Nettoyage v2
PWA offline + FR/EN + PDF A4 + custom types + auto-update

## Deploy (GitHub Pages)
1. Upload all files to your repo root
2. Settings → Pages → Branch: `main` / root → Save
3. Open the Pages URL and test install

## Notes
- Custom types & presets saved in `localStorage` key `est_presets_v4`
- Install prompt requires HTTPS (GitHub Pages/Netlify), not `file://`
